# HiddenConvexityEVNetwork
We present two hidden-convexity reformulations: a broad convex reformulation suitable for
decomposition and pricing, and a special-case SOCP reformulation that serves as a restricted
benchmark.

The code proposes exact and scalable algorithms that exploit the convexified structure, including
integer-lifted OA/Benders cuts and a complete-recourse guarantee.

We design an empirical program to quantify the value of hidden convexity relative to both native
MINLP and decoupled planning baselines.
